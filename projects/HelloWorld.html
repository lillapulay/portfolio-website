Description

Mobile application written in React Native, utilizing React Navigation and React Native Gifted Chat, amongst others. This project was part of the CF Full-Stack Web Development course.

The user can log in and send the following types of messages:

  • text,
  • location,
  • image from camera roll,
  • new photo.

During development I kept a simple KanBan board for managing this project, that included using story points for time estimation.
